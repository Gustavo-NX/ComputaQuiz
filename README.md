O intuito dessa inclusão do projeto é demosntrar um grande projeto que me fez evoluir como desenvolvedor, assim impossibilitando o descarte desse projeto.

Inicialmente este projeto foi efetuado na minha conta O1Gus onde ficava meus repositorios da faculdade, e decidi demosntrar exibir aqui já que foi um projeto que fiquei
muito orgulhoso em participar e feliz com o resultado final, ainda mais na parte de backend onde estava aprendendo do zero e hoje posso melhorar graças a esse projeto.
